# Clone the repo
See [here](https://docs.github.com/en/authentication/connecting-to-github-with-ssh/adding-a-new-ssh-key-to-your-github-account) on how to generate an ssh key and add it to your github account.\
To clone the repo via ssh, use the following command:
```sh
git clone git@github.com:CASHS-CSHS/cashs-cshs.github.io.git
```

To clone via https:
```sh
git clone https://github.com/CASHS-CSHS/cashs-cshs.github.io
```
